## 🎯 Sobre este material

Este repositório contém **100 flashcards** que criei enquanto estudava para o exame da AWS.  
O conteúdo cobre os **4 domínios** da prova:

| Domínio | % do exame |
|---------|-----------|
| Conceitos de nuvem | 26% |
| Segurança e conformidade | 25% |
| Tecnologia | 33% |
| Faturamento e preço | 16% |

Os flashcards complementam meu resumo de estudos e foram feitos com base na documentação oficial da AWS.

## ⚠️ Aviso importante

- Este é um **material de estudo independente**, não oficial da AWS.
- As informações foram coletadas da documentação pública da AWS e de fontes abertas.
- Para garantir sua aprovação, recomendo combinar este material com:
  - Simulados (TutorialsDojo, Udemy, AWS Skill Builder)
  - Documentação oficial da AWS
  - Hands-on no console da AWS
